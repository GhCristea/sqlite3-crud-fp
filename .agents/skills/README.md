# Skills

Skills are domain-specific guidelines loaded on demand when working in a particular area.

## Available Skills

| Skill | When to load |
|---|---|
| `drizzle-orm` | Writing table definitions, queries, migrations |
| `typescript-idioms` | Enforcing `const`-only, `ReadonlyDeep`, no `any` |
| `zod-validation` | Parsing unknown input at I/O boundaries |
| `neverthrow-fp` | Composing `ResultAsync` pipelines, error narrowing |
| `vitest-testing` | Writing unit/integration tests with in-memory SQLite |
